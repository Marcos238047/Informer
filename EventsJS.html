    Cuales son los Eventos HTML y como Funcionan

Apuntes de la clase

Los eventos HTML son muchisimos y pueden servir para aumentar masivamente la interactividad del usuario.

Resumen.

Los eventos HTML son eventos que pueden suceder a los elementos de un HTML, Existen una multitud de ellos que pueden ser utilizados para ejecutar Scripts.

Algunos ejemplos de Eventos: Una pagina termina de cargar, un input fue emitido sobre los elementos como: al hacer clock, al cambiar, al colocar el mouse encima, al precionar una tecla, etc. o mas especificament esi un boton HTML fue presionado.
