Ideas principales

  • Etiqueta Media

    como utilizar la etiqueta CSS @media para definir diferentes estilos para diferentes tipos de medios

Apuntes de la clase

Se pueden combinar el uso de Etiquetas Media y Etiquetas Viewport para crear diseños responsivos facilmente, como definiendo diferentes resoluciones para diferentes pantallas de telefonos celulares.

Resumen.

La etiqueta CSS @Media permite al CSS definir inmediatamente reglas especificas para tipos diferentes de pantallas o medios, como Print para elementos que seran impresos o Screen para pantallas de tipo Computadora, Tablets, Smartphones u otros.

Al utilizar una etiqueta media podemos utilizar diferentes funciones mas alla de la resolucion, como por ejemplo modificar la apariencia del sitio dependiendo de la Orientacion del dispositivo, ETC. por lo que pueden ser extremadamente utilez a la hora de crear dieños web responsivos que no creen inconvenientes o incomodidades al usuario cuando traten de acceder al sitio desde una multitud de formatos.
