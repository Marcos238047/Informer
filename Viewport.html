Etiqueta Viewport

Ideas principales

  • Etiqueta Viewport

    Como utilizar la Etiqueta Viewport para Crear paginas responsivas al dispositivo del usuario

Apuntes de la clase

Los ajustes CSS anteriores se mantienen dentro de la etiqueta viewport a menos que sean cambiados especificamente.

Resumen.

la etiqueta CSS Viewport permite al developer delimitar la apariencia de la pagina a un limite minimo o maximo de resolucion para asi crear diseños que sean convenientes al usuario independientemente desde el dispositivo en el que visite el sitio.

La etiqueta Viewport puede ser utilizada para dar cambios sutiles o totales a la apariencia del sitio o su acomodo ya que las nuevas caracteristicas añadidas a los elementos dentro de un Viewport sobrescribiran a las descritas anteriormente.
